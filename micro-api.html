Servise api on Django¶

Public Api¶

REST API for decks for matching decks by not consistent name

Resource URL:
api/v1/match-deck?name=<deck name>&ship=<ship name>

EXAMPLE:

REQUEST: api/v1/match-deck?name=Three&ship=1
RESPONSE: {deck: Deck 3, sort_order: 3, description: text}
